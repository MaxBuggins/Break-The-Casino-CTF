For my dear group, here is a straight-forward guide to running the CTFd docker:

1. Open your docker compose desktop client.
2. get the terminal to inside the CTFd folder.
3. type in: 'docker compose up' into the terminal.
Example: (my_user@my_pc:~/Downloads/CTFd$ docker compose up

    a. Check available linux distros available : wsl --list --online

    c. You might get an error about a missing WLS update while installing, go on google with the error and you will find the link to the installation of that update.
